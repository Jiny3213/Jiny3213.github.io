重新认识git【其八】stash 临时保存与clean 清理工作区

git stash 临时保存

当你想要切换分支去干别的事情但是又不想提交当前的工作(因为这些工作太琐碎, 你不想占用一个commit), 你就需要临时保存

stash 通常只会保存已跟踪的文件

git clean 清除所有未跟踪文件和空的子目录

有时你需要reset到上一个版本, 但是未追踪的文件还在, 这时候要用clean
